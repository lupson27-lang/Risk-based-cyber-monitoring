| Role                           | Responsibility                                                                | Escalation / Reporting Path                       |
| ------------------------------ | ----------------------------------------------------------------------------- | ------------------------------------------------- |
| CISO / Security Executive      | Approves monitoring strategy, residual risk acceptance, oversight of KPIs     | Reports to Board / Audit Committee                |
| IT Security Manager            | Implements monitoring strategy, coordinates SOC analysts, escalates incidents | Reports to CISO                                   |
| SOC Analyst                    | Monitors alerts, performs initial triage, triggers escalation                 | Reports to IT Security Manager                    |
| System / Application Owner     | Ensures system logs and telemetry are available, coordinates remediation      | Reports to IT Security Manager                    |
| Compliance Officer             | Ensures monitoring aligns with regulatory and audit requirements              | Reports to CISO and Board                         |
| Network / Infrastructure Admin | Maintains network visibility, ensures alerting and logging                    | Reports to IT Security Manager                    |
| OT / ICS Operator              | Monitors process-critical systems, escalates anomalies safely                 | Reports to IT Security Manager and Safety Officer |
